## ðŸš€ Usage Guide

Use the following commands in your terminal to clone the repository and begin:

### Clone the Repository

# Clone the repository to your local machine
git clone [repository-url] auth-16-ui-kit

# Navigate into the new directory
cd auth-16-ui-kit
Integrate the Library
The preferred method is loading the .excalidrawlib file directly into Excalidraw:

Open the Excalidraw application.

Click the "Library" panel icon (or press 9).

Click "Load" and select the auth-16.excalidrawlib file from your local cloned directory.

The 16 prefabs will now be available in your library for immediate use.

### Alternative: Using the Demo Page
Open the local index.html file in your web browser.

Click the "Add 16 Components to Workspace" button on the page to instantly inject the collection into a new Excalidraw session.
