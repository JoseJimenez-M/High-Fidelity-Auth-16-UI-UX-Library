Welcome to the repository for the **Auth 16 UI Kit**, an essential collection of **16 high-fidelity components** designed to streamline the creation of user authentication and account management flows in [Excalidraw](https://excalidraw.com/).

This kit enables you to quickly prototype critical user journeys with a clean, professional design, avoiding the need to start from scratch.

## üì¶ What's Included

This repository provides the core assets needed to immediately use this 16-component library:

1.  **`auth-16.excalidrawlib`**
    * Contains all **16 high-fidelity prefabs** that cover the entire user account lifecycle.
    * Elements share a unified aesthetic (based on a violet/purple theme) for visual consistency across your diagrams.

2.  **`index.html`**
    * A custom landing page for showcasing and previewing the 16 prefabs.
    * Simplifies integration by allowing you to instantly load the entire collection into an Excalidraw board via a dedicated button.

## üöÄ Usage Guide

### The preferred method is loading the .excalidrawlib file directly into Excalidraw:

Open the Excalidraw application.

Click the "Library" panel icon (or press 9).

Click "Load" and select the High-Fidelity Auth 16 UI UX Library.excalidrawlib file.

The 16 prefabs will now be available in your library for immediate use.

### Alternative: Using the Demo Page
Open the local index.html file in your web browser.

Click the "Add 16 Components to Workspace" button on the page to instantly inject the collection into a new Excalidraw session.
